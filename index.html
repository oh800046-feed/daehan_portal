대시보드 사용 안내

1 구획(섹션) 순서 바꾸기

각 구획 제목 왼쪽의 손잡이 ⠿를 마우스로 잡아 다른 구획 위로 끌어다 놓으면 구획 순서가 바뀝니다. (경영진 · 기획담당 · 회의체)

2 아이콘 위치/구획 이동

아이콘에 마우스를 올리면 위쪽에 손잡이 ⠿가 나타납니다. 이를 잡아 다른 아이콘 위나 다른 구획·그룹으로 끌면 위치가 이동합니다. 빈 영역에 놓으면 해당 그룹 맨 뒤로 들어갑니다.

3 이름·설명 수정

아이콘의 이름 또는 설명 글자를 클릭하면 그 자리에서 바로 편집됩니다. 입력 후 Enter나 다른 곳을 클릭하면 저장됩니다.

4 연동 주소 확인

아이콘 위에 마우스를 올리면 마우스를 따라다니는 말풍선에 🔗 연동 웹주소가 표시됩니다. 주소가 없는 항목은 준비중으로 안내됩니다. 클릭하면 새 탭에서 해당 사이트가 열립니다.

5 이미지 삽입

아이콘 오른쪽 위 🖼 버튼을 누르면 이미지 파일을 선택할 수 있고, 선택한 이미지가 아이콘 자리에 채워집니다. 실제 로고나 캡처로 교체할 때 사용하세요. (작은 이미지를 권장합니다.)

6 프로그램 추가 · 삭제

오른쪽 위 + 추가로 이름·설명·URL·아이콘 글자·구획·색상을 지정해 새 항목을 만듭니다. 삭제는 관리 버튼을 눌러 편집 모드로 전환한 뒤, 각 아이콘 왼쪽 위 × 버튼으로 삭제합니다.

7 화면 구성 · 크기 · 테마

노트북 / 와이드 / 전체 버튼으로 화면 폭에 맞춰 배치를 최적화하고, 크기 슬라이더(60~130%)로 아이콘 전체를 키우거나 줄일 수 있습니다. 다크/라이트 테마도 전환됩니다.

8 인쇄

상단 🖨 인쇄 버튼을 누르면 현재 화면 구성 그대로 인쇄(또는 PDF로 저장)할 수 있습니다. 인쇄 시에는 편집용 버튼과 손잡이는 자동으로 숨겨집니다.

변경한 배치·이름·이미지는 새로고침하면 초기화됩니다. 영구 저장이 필요하면 관리자에게 저장 기능 추가를 요청하세요.
